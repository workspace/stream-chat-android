Basic implementation of ChannelEventsHandler. It handles basic channel events like NotificationAddedToChannelEvent, ChannelDeletedEvent, NotificationChannelDeletedEvent, ChannelUpdatedByUserEvent, ChannelUpdatedEvent. It skips other type of events, mark as remove result EventHandlingResult.REMOVE for deleted events, other logic you're free to implement.
